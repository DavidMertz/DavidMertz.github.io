Directory listing of http://gnosis.cx/publish/resumes/

Older resumes are available as source code in one of several scripting languages. The cute idea is to include just enough concise code in each version to make the document sensitive to unauthorized modifications (as head-hunters are wont to). Run each version as indicated within the document, and it will report on its own authenticity.
